## 📈 Prompt Evaluation Criteria

Each prompt in this library was evaluated and improved through multiple iterations (V1 → V3) based on the following criteria:

### 1. Relevance
The extent to which the prompt aligns with the specific HR workflow task and business context.  
Prompts must directly support recruitment or onboarding activities for a Social Media Head in an online skincare brand.

### 2. Clarity
The prompt should clearly define the task, expected output, and role context.  
Ambiguity is minimized by specifying instructions, constraints, and expected structure.

### 3. Accuracy
Outputs must be based strictly on the provided input (e.g., resumes, interview notes) to avoid incorrect assumptions or hallucinations.  
Prompts include instructions such as “avoid assumptions” and “use only provided information.”

### 4. Actionability
The output generated by the prompt should be directly usable in a real business setting.  
For example, emails should be ready to send, and evaluations should support decision-making.

### 5. Structure
Prompts are designed to produce well-organised outputs (e.g., summaries, tables, evaluations).  
Structured outputs improve readability and consistency across HR processes.

### 6. Constraint Design
Effective prompts include constraints such as word limits, tone, format, and scope.  
This ensures consistent, professional, and high-quality outputs.

### 7. Automation Suitability
Each prompt is evaluated based on how effectively it can automate or support a task.  
Tasks that are repetitive and text-based have higher automation potential.

### 8. Risk Awareness
Prompts are assessed for potential risks such as bias, hallucination, or over-reliance on AI.  
Mitigation strategies include human review and evidence-based evaluation.

## 🔁 Iterative Improvement Approach
All prompts were refined across three versions:
- **V1:** Basic prompt with minimal structure  
- **V2:** Improved clarity and task definition  
- **V3:** Fully optimised with constraints, structure, and business alignment  

This iterative process ensured continuous improvement in prompt quality and reliability.
